トークスクリプト


【オープニング】（約1分）


「前回スクリプトの概念を学びました。今回は最初のスクリプトとして、ファイルの一括リネームと整理スクリプトを実際に作ります。


写真・書類・ダウンロードファイルの整理で毎回時間を使っている方に特に役立つ内容です。」



【本編①：作るスクリプトの仕様】（約2分）


「今回作るスクリプトの仕様を決めます。


スクリプト名： organize_files.py


機能1：ファイルの仕分け整理


指定したフォルダ内のファイルを拡張子ごとにサブフォルダに仕分けします。


  • .jpg .png .gif → Images/フォルダ
  • .pdf → PDFs/フォルダ
  • .docx .doc → Documents/フォルダ
  • .xlsx .csv → Spreadsheets/フォルダ
  • その他 → Others/フォルダ

    • 機能2：ファイルのリネーム


    ファイル名の先頭に今日の日付を追加します。


    例：sample.pdf → 2025-07-15_sample.pdf



    この2つの機能を持つスクリプトを作ります。実行すると指定フォルダが整理されます。」



    【本編②：実演：スクリプトを作って動かす】（約6分）


    「【画面収録開始】


    まずテスト用のフォルダにバラバラなファイルを用意します。写真・PDF・Word・Excelが混在している状態です。



    チャット欄に指示を入力します。


    『以下の仕様でファイル整理スクリプトを作成してください。


    ファイル名：organize_files.py


    機能：

  • コマンドライン引数で整理対象のフォルダパスを受け取る
  • フォルダ内のファイルを拡張子ごとにサブフォルダに移動する

    • - .jpg .png .gif .webp → Images/

    - .pdf → PDFs/

    - .docx .doc .txt → Documents/

    - .xlsx .xls .csv → Spreadsheets/

    - その他 → Others/

  • 移動前にファイル名の先頭に今日の日付（YYYY-MM-DD形式）を追加する
  • 処理したファイル数と移動先をターミナルに表示する
  • 対象フォルダが存在しない場合はエラーメッセージを表示する

    • コメントはすべて日本語で書く』



    スクリプトが生成されました。


    テスト実行をします。


    『デスクトップの test_folder を整理してください。このスクリプトを使って実行してください』


    実行後、test_folderを確認すると、Images・PDFs・Documentsサブフォルダに仕分けられています。ファイル名の先頭にも日付が追加されています。



    スクリプトの内容を少し見てみましょう。コードが表示されますが、全部理解する必要はありません。コメントが日本語で書かれているので、処理の流れはなんとなくわかります。


    【画面収録終了】」



    【クロージング】（約1分）

  • 次回は実行をさらに簡単にする方法を学ぶ

    #### 【オープニング】 0:00〜1:00

  • デスクトップのrename_testフォルダを開いて「このバラバラなファイルを一括リネームするスクリプトを作ります」と映す

    • #### 【本編①：作るスクリプトの仕様】 1:00〜3:00

  • 「ルール：全ファイルを2024-01-01_01.jpg形式に変換したい」と口頭で説明

    • #### 【本編②：実演 スクリプトを作って動かす】 3:00〜9:00

  • **3:00〜3:30** 「指定フォルダ内のファイルを日付_連番.拡張子形式でリネームするPythonスクリプトを作ってください」と入力してEnter
  • **3:30〜5:00** 承認でYes → rename.pyが生成される → エディタで中身をチラ見せ
  • **5:00〜5:30** 「python rename.py /Users/自分のユーザー名/Desktop/rename_test と実行してください」と入力 → Enter
  • **5:30〜6:30** 承認でYes → 実行
  • **6:30〜8:00** Finderでrename_testフォルダを開く → ファイル名が変わっていることを映す（Before/After）
  • **8:00〜9:00** 「10個のファイルが一瞬で変わりました！これがスクリプトの力です」と解説

    • #### 【クロージング】 9:00〜10:00

  • 「次回はCSV集計スクリプトを作ります」と締める
