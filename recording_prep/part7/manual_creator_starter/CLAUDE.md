# マニュアル自動生成アプリ

## プロジェクト概要
動画ファイルを読み込んで、ffmpegでスクリーンショットを切り出し、
Claude APIで各画像の説明文を生成して、画像つきステップマニュアルとして
出力するデスクトップアプリです。

## 使用技術
- Python 3.11
- ffmpeg（動画から画像切り出し）
- Claude API / anthropic SDK（画像認識・説明文生成）
- tkinter（デスクトップGUI）
- python-docx（Word出力）
- reportlab または weasyprint（PDF出力）

## フォルダ構成
- /src：アプリ本体のPythonコード
- /output：生成されたマニュアルの保存先
- /temp：処理中の一時ファイル（切り出し画像など）
