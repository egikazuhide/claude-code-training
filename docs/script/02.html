【本編①：3種類のバージョンとデスクトップ版を使う理由】（約2分）

【画面：ブラウザでclaude.aiを開き、ページを見せながら説明】


「まず知っておきたいのが、Claude Codeには3つのバージョンがあるということです。


1つ目はブラウザ版。

今見えているこのページがブラウザ版です。URLを開くだけですぐ使えますが、ファイルをプロジェクトとして管理する機能に制限があります。


2つ目はCLI版。

CLIとはコマンドラインインターフェースの略で、黒い画面に文字を打ち込んで操作する形式です。エンジニア向けで、初心者にはハードルが高いです。


3つ目がデスクトップアプリ版。

パソコンにインストールして使う形式です。この講座では全編このデスクトップ版のみを使います。


デスクトップ版を選ぶ理由は3つです。

  • インストールするだけでAPIキーなどの細かい設定が不要
  • 画面を見ながら視覚的に操作できる
  • ファイルをドラッグ&ドロップで渡せる

    • 初心者の方には最もとっつきやすい形式です。」



    【本編②：Macのインストール手順】（約3分）

    【画面：Macのデスクトップ。Windowsの方はこのブロックを飛ばしてください】


    「ではMacの方から進めます。Windowsの方はこのブロックを飛ばして次のブロックから見てください。


    まずブラウザを開いてください。URLバーに『claude.ai』と入力してEnterを押します。


    Claudeのトップページが開きます。ページ内の『ダウンロード』リンクをクリックして、『Mac用をダウンロード』ボタンをクリックします。


    ダウンロードが完了したら、ダウンロードフォルダを開いてください。『Claude.dmg』というファイルがあります。ダブルクリックして開きます。


    インストーラーが起動してClaudeのアイコンが表示されます。アイコンをApplicationsフォルダにドラッグ&ドロップしてください。これでインストール完了です。



    ここで注意点があります。


    はじめて起動しようとすると『このアプリはインターネットからダウンロードされました』という警告が表示されることがあります。これは正常です。『開く』ボタンをクリックして進んでください。


    もし『開けません』と表示された場合は、システム設定を開いて、プライバシーとセキュリティから『このまま開く』を選べば起動できます。」



    【本編③：Windowsのインストール手順】（約3分）

    【画面：Windowsのデスクトップ。Macの方はこのブロックを飛ばしてください】


    「次にWindowsの方の手順です。Macの方はこのブロックを飛ばして次に進んでください。


    ブラウザを開いてURLバーに『claude.ai』と入力してEnterを押します。


    トップページが開いたら、ダウンロードリンクをクリックし、『Windows用をダウンロード』をクリックします。


    ダウンロードが完了したら、ダウンロードフォルダ内の『ClaudeSetup.exe』をダブルクリックします。


    『このアプリがデバイスに変更を加えることを許可しますか？』という確認が出ます。『はい』をクリックしてください。


    インストールが自動で進みます。完了するとデスクトップにClaudeのアイコンが作成されます。



    Windowsでの注意点をお伝えします。


    セキュリティソフトが反応することがあります。その場合は一時的に許可するか、セキュリティソフトの設定で例外登録してください。


    インストール先はデフォルトのまま変えないことをおすすめします。」



    【本編④：アカウント登録・ログイン・初期設定】（約2分）

    【画面：Claude Codeデスクトップアプリを起動した直後のログイン画面】


    「Claude Codeを起動するとこのようなログイン画面が表示されます。


    まだアカウントを持っていない方は『アカウントを作成』をクリックしてください。メールアドレスとパスワードを登録するだけで完了します。Googleアカウントでのログインも可能です。


    すでにアカウントをお持ちの方はそのままログインしてください。



    ログインが完了するとClaude Codeのホーム画面が開きます。


    この段階でやっておくべき初期設定が2つあります。


    1つ目は表示言語の確認です。設定アイコンをクリックして、言語が日本語になっているか確認してください。


    2つ目はテーマの設定です。画面が明るすぎる・暗すぎる場合は、同じ設定画面でライトモード・ダークモードを切り替えられます。


    これで起動の準備は完了です。」

    操作・画面の流れ（詳細版）


    📋 収録前準備

  • [ ] ブラウザでclaude.aiを開いておく
  • [ ] Macの場合：Finderを開いておく
  • [ ] Windowsの場合：エクスプローラーを開いておく
  • [ ] ダウンロード済みのインストーラーをデスクトップに用意（または実際にダウンロードする）

    • 🎬 収録手順


    #### 【オープニング】（0:00〜1:00）

  • 音声のみで「今日はClaude Codeをインストールします」と話す

    • #### 【本編①：ダウンロード（Mac）】（1:00〜4:00）

  • ブラウザでclaude.aiを開く → 画面を映す
  • ダウンロードボタンを探してクリック（またはすでにあるdmgファイルを見せる）
  • ダウンロードフォルダにdmgファイルが届いたことを確認
  • dmgファイルをダブルクリック → マウントされるアニメーションを映す
  • Claude Codeアイコンを「Applications」フォルダにドラッグ → 映す
  • 「セキュリティ警告が出ても問題ありません」と口頭で説明しながら確認ダイアログでOKをクリック

    • #### 【本編②：ダウンロード（Windows）】（4:00〜7:00）

  • Windowsの場合はexeファイルのダウンロード → ダブルクリック → インストールウィザードの流れを映す
  • 「次へ」「インストール」「完了」をクリックする様子を見せる

    • #### 【本編③：ログイン】（7:00〜11:00）

  • Claude Codeアプリを起動（Launchpad or スタートメニューから）
  • ログイン画面を映す
  • 「Anthropicアカウントでログイン」ボタンをクリック
  • ブラウザが開いてAnthropicのログインページが表示される
  • メールアドレスとパスワードを入力してログイン（ここは素早く映してOK）
  • アプリに戻ると「ようこそ」画面が表示される → 映す

    • #### 【クロージング】（11:00〜12:00）

  • 「これで準備完了です」と音声で締める

    • ⚠️ 収録時の注意点

  • パスワード入力時は画面をぼかすか、すでにログイン済みの状態から見せる
  • Macの場合のセキュリティ警告は必ず見せること（視聴者が驚かないよう）
