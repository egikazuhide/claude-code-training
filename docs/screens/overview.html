manual_creator - アプリ全体像
完成イメージ：「マニュアル自動生成アプリ」

操作を録画した動画ファイルを読み込ませるだけで、画像つきの手順マニュアル（Word / PDF）を自動で作成するデスクトップアプリ

🎬
動画ファイル
操作を録画した.mp4
→
🖼️
画像を切り出し
ffmpegで一定間隔ごとに静止画を抽出
→
🤖
AIが説明文を生成
Claude APIが各画像の操作内容を文章化
📋
プレビュー画面
Step1, 2, 3...と並べて表示
編集・削除・並び替え
→
📄
Word / PDF 出力
画像つき手順マニュアルとして保存
