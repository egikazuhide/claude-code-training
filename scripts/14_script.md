### 【本編①：作るスクリプトの仕様】（約2分）

「まずは設計図、ちゃんと決めますよ。行き当たりばったりでAIに投げると、いい結果は出ません。

**スクリプト名：** organize_files.py

**機能1：ファイルの仕分け整理**

指定したフォルダ内のファイルを『拡張子（ファイル名の末尾につく.jpgや.pdfみたいな種類の印）』ごとにサブフォルダに仕分けします。

- .jpg .png .gif → Images/フォルダ
- .pdf → PDFs/フォルダ
- .docx .doc → Documents/フォルダ
- .xlsx .csv → Spreadsheets/フォルダ
- その他 → Others/フォルダ

**機能2：ファイルのリネーム**

ファイル名の先頭に今日の日付を追加します。

例：sample.pdf → 2025-07-15_sample.pdf

---

はい、この2つの機能、欲張りセットでいきます。実行すれば指定フォルダが一瞬で生まれ変わる、そんなスクリプトを作っていきましょう。」

チャット欄に指示を入力していきます。仕様、ちゃんと日本語でぶつけるだけです。

『以下の仕様でファイル整理スクリプトを作成してください。

ファイル名：organize_files.py

機能：
1. コマンドライン引数で整理対象のフォルダパスを受け取る
2. フォルダ内のファイルを拡張子ごとにサブフォルダに移動する
   - .jpg .png .gif .webp → Images/
   - .pdf → PDFs/
   - .docx .doc .txt → Documents/
   - .xlsx .xls .csv → Spreadsheets/
   - その他 → Others/
3. 移動前にファイル名の先頭に今日の日付（YYYY-MM-DD形式）を追加する
4. 処理したファイル数と移動先をターミナルに表示する
5. 対象フォルダが存在しない場合はエラーメッセージを表示する

コメントはすべて日本語で書く』

### 🎬 収録手順（10分完全版）

- デスクトップのrename_testフォルダを開いて「このバラバラなファイルを一括リネームするスクリプトを作ります」と映す

#### 【本編①：作るスクリプトの仕様】 1:00〜3:00
- 「ルール：全ファイルを2024-01-01_01.jpg形式に変換したい」と口頭で説明

#### 【本編②：実演 スクリプトを作って動かす】 3:00〜9:00

4. **5:30〜6:30** 承認でYes → 実行
5. **6:30〜8:00** Finderでrename_testフォルダを開く → ファイル名が変わっていることを映す（Before/After）
